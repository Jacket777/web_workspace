1.条件语句基本使用

1.1.条件语句条件：一般需要的是布尔值，但是其他类型数值也可以参与（隐式转换为布尔值）
1.2.如果条件语句只有一行代码可以省略大花括号

2.if结合else关键字一起使用

2.1.else关键字可以结合if条件语句一起使用（可以：不是必须）

2.2.else这个分支对于上面分支条件进行否定

实例代码

var score = prompt("请你输入你的分数");
if (score >= 60) {
//条件：分数大于等于60
alert("兄弟恭喜你及格啦");
} else {
//暗含条件：对于上面条件进行否定（分数是小于60分）
alert("兄弟不好意思,来年回来继续考试，我等你");
}
